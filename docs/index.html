Python module for user interation with assignment 3 package based off the terminal

ROS node which gives the user a series of choices for interaction with the assignment 3 package between

  • Option 1 (1): giving desried coordinates

  • Option 2 (2): Manual Control without assistance

  • Option 3 (3): Manual Control with assistance

  • Option 0 (0): exit

scripts.teleopContrNode.callback_scan(data)[source]

This is a callback function called when the topic /scan is subscribed to. It creates a publisher object to publish to the /cmd_vel topic for controlling the robot movement, and determines whether the right, forward, or left directions are obstructed by a wall by using the compute_min_dist function to compare the distance of the sectors to a min_dist variable (0.5 in this case).

Parameters

data (obj) – Instance of the Laser scan class.

scripts.teleopContrNode.compute_min_dist(ranges)[source]

This function uses the data gotten from the laser scanner attachment to compute the minimum distance observed indicating the presence of a wall in a certain direction. Where the direction within the range of 0 to 720 degrees is given as: - right (0-240) - forward (241-480) - left (481-721)

Parameters

ranges (list) – List of 720 float values, of range values obtained from the LaserScan topic.

Returns

This is the shortest distance from the robot scanned by the laser scanner sensor.

Return type

distance (float)
